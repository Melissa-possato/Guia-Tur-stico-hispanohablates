# Guia-Turistico-hispanohablates

Passo a Passo para executar o programa em cada máquina:
* Para esse programa foram utilizados MySQL Workbench e VSCODE, é fundamental que ambos os softwares estejam presentes na máquina para o funcionamento do programa.

1 - No GitHub, clone o URL do projeto para sua área de transferência;
2 - Também no GitHub, salve o arquivo "guiaTuristico.sql";
3 - Abra o MySQL Workbench;
4 - Entre em uma conexão;
5 - Abra o arquivo "guiaTuristico.sql" em sua query 1, rode e salve o arquivo.
2 - Abra o VSCODE na máquina;
3 - Abra uma pasta onde será executado o programa;
4 - Abra um novo terminal e escreva "git clone " em seguida cole a URL do projeto;
5 - Certifique-se que foi clonado corretamente todas as pastas do projeto para seu gerenciador de arquivos;
6 - Caso não tenha clonado corretamente apague a pasta do guia turístico e repita o processo 4;
7 - Clonado corretamente, abra a pasta que foi clonada no seu VSCODE;
8 - Em seguida, abra a pasta "backend" e em seguida o arquivo "db.js";
9 - Certifique-se que o "user", "password", "port" estejam com as mesmas informações da sua máquina, caso necessário, altere as no arquivo e salve novamente (Ctrl S);
10 - Abra o MySQL Worchbenck
11 - Entre em sua conexão
